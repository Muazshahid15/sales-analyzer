# 🏆 Sales Performance Analyzer

A complete end-to-end data analytics project built in Python, covering data generation, cleaning, exploratory analysis, visualization, statistics, and reporting.

---

## 📌 Project Overview

This project analyzes 1,000 simulated B2B sales records across regions, products, and sales reps for the year 2023. The goal is to surface actionable business insights: who are the top performers, which products drive the most revenue, and where are the biggest opportunities?

---

## 🛠️ Tech Stack

| Tool | Purpose |
|---|---|
| Python 3 | Core language |
| Pandas | Data manipulation |
| NumPy | Numerical computing |
| Matplotlib & Seaborn | Visualization |
| Scikit-learn | Linear regression model |
| SciPy | Statistical analysis |
| OpenPyXL | Excel export |
| Jupyter Notebook | Development environment |

## 🚀 How to Run

1. Clone the repo
```bash
   git clone https://github.com/Muazshahid15/sales-analyzer.git
   cd sales-analyzer
```

2. Install dependencies
```bash
   pip install -r requirements.txt
```

3. Launch Jupyter
```bash
   jupyter notebook
```

4. Open `sales_analysis.ipynb` and run all cells top to bottom
